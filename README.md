## 操作系统作业调度算法模拟

+ 说明：操作系统实验作业，实现常见的几种作业调度算法，如FCFS(先来先服务)、SJF(短作业优先)、HRRN(高响应比优先)、SRT(最短剩余时间)等等。
+ 实现约束：

  + 调用STL模板库中的算法及数据结构
+ 不能自己实现算法
  + 不能自己实现数据结构
+ 采用分离式开发，源文件与头文件分离
